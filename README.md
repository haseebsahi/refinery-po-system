# RPOS – Refinery PO System

A web-based Purchase Order management system for refinery procurement, built with React, TypeScript, Tailwind CSS, and Supabase.

## Features

- Browse and search a catalog of refinery parts & equipment
- Create and manage Purchase Orders with line items
- Track PO status (Draft → Submitted → Approved → Fulfilled)
- Real-time backend integration

## Getting Started

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd <YOUR_PROJECT_NAME>

# Install dependencies
npm i

# Start the development server
npm run dev
```
